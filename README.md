
# checkrpkgs

With `checkrpkgs`, you can get information on installing and configuring
R, RStudio, and Rtools; setting up and using Git and GitHub; obtaining
information about to-be-installed and already-installed packages.

## Installation

You can install `checkrpkgs` from
[GitHub](https://github.com/JesseAlderliesten/checkrpkgs) with:

``` r
if(!requireNamespace("remotes")) {
  install.packages("remotes")
}
remotes::install_github("JesseAlderliesten/checkrpkgs")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(checkrpkgs)
## basic example code
```
